# Gallery Images

This folder contains curated images for the homepage gallery.

## Structure
Each image should have a corresponding entry in `gallery-data.json`.

## Categories
- **render**: Photorealistic renders, exterior/interior views
- **ideation**: Operative massing, volumetric studies
- **sketch**: Hand-drawn sketches, conceptual drawings
- **diagram**: Analytical diagrams, plan drawings
- **presentation**: Final presentation boards

## Adding New Images
1. Save your generated image to this folder (e.g., `my-render-01.jpg`)
2. Add an entry to `gallery-data.json` with metadata
3. Push to GitHub to update the website
